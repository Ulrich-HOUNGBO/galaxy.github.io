GALAXY GAME

Project to create a cross-platform game using the Kivy Framework

GALAXY GAME RULES

The Galaxy project is a perspective game that displays a terrain on which the user must move left and right. It's a game that can run on PC, Mac, Linux and also on mobile devices, iOS and Android.. If you leave the field, you've lost. When you click on the keys for part two, the terrain will be generated randomly and infinitely. and infinitely variable. We'll display our ship and manage collisions. In other words, as long as the ship is on the terrain, everything's fine. But as soon as it leaves the terrain, we'll go into game over and display this game over trace.
We chose to carry out this project with the aim of acquiring new skills. With this project we set ourselves the objectives of being able to create cross platform applications Windows Mac OS, Android IOS and also practice on advanced algorithms with random and unlimited terrain generation and projection to display terrain in perspective.
